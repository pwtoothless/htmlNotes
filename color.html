There are 3 ways to do Color
- name
- Hex
- Styles using RGBA

Name is limited 140 defined names that you can lookup

If you use Hex you have 16,777,216 colors
- Syntax you are going to use a # in front of the color
- Web Safe colors use repeated values

RGBA Which stands for: Red, Green, Blue, Alpha - Can only be used with stylesheets
- Syntax will be rgba(0-255, 0-255, 0-255, 0-1)
